# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, email [ukorotovskiy@gmail.com](mailto:ukorotovskiy@gmail.com) with the details, affected versions, and reproduction steps if available.  
We will respond promptly and work with you to address the issue responsibly.

Thank you for helping keep this project and its users safe!
